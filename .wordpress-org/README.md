# WordPress.org Assets

This directory contains assets for the WordPress.org plugin repository.

## Screenshots

Add PNG screenshots here with the following naming:
- `screenshot-1.png` - Product page shipping calculator
- `screenshot-2.png` - Plugin settings page  
- `screenshot-3.png` - Orders and label management
- `screenshot-4.png` - Balance and PIX payment
- `screenshot-5.png` - Debug logs interface

**Recommended size:** 1280x720 pixels (16:9 aspect ratio)
